



# Learn japanese letters

This is a website to learn japanese letters hiragana and katakana.
You can hear the letters sound
## What I used:
- HTML
- CSS
- little javascript
## How it works
- open the website
- choose hiragana or katakana
- click on any letter to hear it
### I used AI in only :
When clicking on any letter , the letter sound will play , all other things have been created by me
